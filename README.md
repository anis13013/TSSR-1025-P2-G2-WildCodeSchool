# TSSR-1025-P2-G2-WildCodeSchool
Script unifié d'administration réseau permettant de découvrir automatiquement les machines actives sur un réseau local (172.16.20.0/24), d'identifier leur OS (Windows/Linux) et d'exécuter des tâches d'administration à distance via SSH — gestion des comptes, du stockage, du réseau, des services et de la sécurité.
